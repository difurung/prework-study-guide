HTML

  • HTML provides the structure for the webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to to the user.
  • The meta element defines the metadata such as the character set, descrioption, keywords, author, and viewport
  • The satyle element defines the CSS styles associated with the website.
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document.

    CSS

  • CSS stands for Cascading Style Sheets.
  • CSS adds style and color to the existing HTML framework.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • The most common way to style a webpage is by using an external CSS style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • Multiple declarations are called a declaration block.
  • We use the link element in HTML to link the CSS style sheet to the HTML file.
